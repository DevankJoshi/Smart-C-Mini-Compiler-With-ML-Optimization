10 FEATURES EXTRACTED FROM CODE
═════════════════════════════════════════════════════════════════════════════

1. instruction_count       - Total TAC instructions
2. temporaries            - Temporary variables used
3. arithmetic_ops         - Arithmetic operations
4. memory_access          - Load/store operations
5. branch_count           - Conditional branches
6. loop_count             - Loop structures
7. nesting_depth          - Max loop nesting level
8. repeated_expressions   - Duplicate computations
9. constant_expressions   - Compile-time constants
10. dead_assignments      - Unused variable assignments


6 OPTIMIZATION CLASSES
═════════════════════════════════════════════════════════════════════════════

Class 0: None                        - No optimization needed
Class 1: Constant Folding            - Precompute constants
Class 2: Dead Code Elimination       - Remove unused code
Class 3: Common Subexpression Elim.  - Reuse computations
Class 4: Loop Invariant Code Motion  - Extract from loops
Class 5: Loop Unrolling              - Unroll tight loops
